This repository hosts the K-PROTOCOL analysis engine, which extracts 15 years of pulsar timing data from mainstream academia (NANOGrav) to demonstrate **absolute speed of light decay ($\Delta c$)** and the **Earth's geometric distortion coefficient ($S_{earth}$)**.

It visually and perfectly proves through 1.1 million actual cosmic data points that the speed of light ($c$), which mainstream physics has considered a 'constant', is actually decaying at a rate of **0.0023 m/s** per year.

### 🔍 Core Master Formula
`Geometric Delay (ns) = (Years Elapsed * 0.0023 / 297,880,197.6) * 1.006419562 * 10^9`
